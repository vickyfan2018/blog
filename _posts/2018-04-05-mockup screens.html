Responsive logo

I designed the original logo two years ago. Although I do not like this logo now, it has registered in the government already, so I still need to use this logo. When designing this logo two years ago, I did not design it with web application in mind, so I tried to simplify the logo to make it suitable being displayed on web.


Moodboard and branding elements

As Chun Kee do not have any branding elements, I need to remake every visual elements showcasing the brand images, so I made a moodboard first.

As Chun Kee is a traditional Chinese brand, which still used bamboo steamers, so the color patlette is brownish with some red tone, showcasing that it is a vintage Chinese noodle shop. To show the original and vintage images of Chun Kee, hand-drawn illustrations showing different noodle and imprinted typefaces are used.
