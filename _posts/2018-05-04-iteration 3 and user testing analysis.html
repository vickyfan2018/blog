User testing

User interviews

I conducted user interviews with 5 users (in Cantonese) to improve my website. Here is the questions.

    • How do you feel about the website?
      What do you think about the index page?
      How do you feel about the progress in accessing the product page?
      How do you feel about the flow of the website?

    • As the transcript of the interviews is in Chinese, I am not going to post and rewrite the transcript in English, but after the user interviews, I found several main comments.

    • The website is really easy to navigate. Users can go to the product page easily.
      The website creates a vintage mood and successfully conveys the story of a shop with long history.
      The video on the index page attracts users' attention.

    • User testing - observation

    I tried to do live observation when users accessing the webpage. Thus, I asked 5 users to go to the site, and I observed their activities on mouseflow.com.

    I found that when doing user testing all of the users sucessfully went into the product page and they accessed to the highlighted and the most expensive products, which were good. However, I found how they use the website is different from people who use the website remotely. Because they knew that I was watching them using the website when they navigated the website, they tried to click and scroll more. However, without telling users I was watching them using the site, they didn't scroll and click as frequent in observations. Hence, I used remote user testing instead.

    Remote user testing

    Relooping the video

    Some classmates suggested me to delete the reloop button and make the video relooping by itself. However, I found that half of the users have clicked this button. I'm still thinking about if I should delete the button.

    Navigation bar

    I found that users tend to click the about us button on navigation bar, rather than the more button in the about us section in the index

    Scrolling percentage

    I found that users only scroll through 50% to 70% of the pages. Especially, in the index page, they only scrolled to the middle of the page, which means that they only saw the top of the page and product highlighted section. Hence, this means that the about us section is ignored, so this section in the index is quite useless.
