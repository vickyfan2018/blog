Link to the website

Iterations based on user feedback

Favicon

To make the website more professional, I added a favicon to the website


Gif to mp4

Although some classmates suggest that I could change the gif in the opening screen to a mp4 file, after I tested it with my user, the mp4 file could not be loaded in the mobile version, so I keep using the gif file



Increasing space on the top of the "About Us" section

In the previous version, the top of the "Contact Us" section was cut after the user clicked the "Contact Us" on the navigation menu.

Therefore, I added this snippet of code.

Hence, there is enough space between the top of the section and the text.


Increasing the opacity of the text

Some users said it is hard to see the text in the half transparent black background.

Therefore, I deleted the black wrapper of the text, and added an overlay on the background image, so the opacity of the text would not be affected by the black background's opacity.

Hence, the text above the photos is more legible, and I also added text shadow to increase the legibility.

Increasing the section height and font size

As there are more users accessing the page by desktop, I increased the font size and the height of some sections, so that the text in each section will not be overlayed and the text is more legible in desktop version.
